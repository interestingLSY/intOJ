评测
本OJ搭建在Ubuntu 18.10(64位)上,为Linux环境. 编译命令: C++: `g++ -DONLINE_JUDGE {src_file} -o {exec_file} -lm -w` C++11: `g++ -std=c++11 -DONLINE_JUDGE {src_file} -o {exec_file} -lm -w` 请注意 `-w` 代表编译器将不输出任何警告信息 请使用标准输入输出
评测状态
| 状态 | 全称 | 中文解释 | | :-------: | :--------: | :--------: | | AC | Accept | ok,你的程序通过了! | | CE | Compile Error | 编译错误 | | WA | Wrong Answer | 答案错误 | | TLE | Time Limit Exceed | 时间超限 | | MLE | Memory Limit Exceed | 内存超限 | | RE | Runtime Error | 运行时错误 | | PE | Presentation Error | 格式错误 | | OLE | Output Limit Exceed | 输出超限 | | UKE | Unknown Error | 未知错误...评测出锅了 |
